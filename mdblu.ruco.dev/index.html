📄

Purpose-built templates

MISSION, SPEC, ADR, HANDOFF — each structured so Claude can reason about your project without open-ended interpretation.

🔧

MCP server

use_template, propose_template_update, scaffold_hook — Claude scaffolds, reads, and proposes updates through the MCP protocol.

🔁

Auto-improvement

mdblu detects when a template could be improved based on usage patterns and opens a PR automatically — the system evolves as you work.

🧩

Composable

Drop any template into any project. They work standalone or as a full system. No lock-in to a specific workflow or tool stack.

Get started

Scaffold a document in the current directory.

mdblu use MISSION
